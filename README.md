GuessPic is a multiplayer drawing guessing game. This project was aim to practice Websocket, allowing the action to be sync accross all users in the room.

## Features
- **Create/Join Room**: User have ability to create a room and each room will be attatch with an unique ID, other users can join the same room by inputting the roomId
- **Draw/Guess**: Every round, a player in the room will be picked to be the drawer and everyone else have to 60 seconds to guess what the drawer is drawing.

## Technologies
- **Programming Language**: Typescript, Python
- **Framwork**: React.js, FastAPI, WebSocket, TailwindCSS
